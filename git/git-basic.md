
## Git Basic

1. $ git init 
2. $ git clone username@host:/remote/repository/path
3. $ git add <file name>
4. $ git commit -m "commit message"
5. $ git push origin master

<br/>
## Git Branch

* $ git checkout -b feature_x
* $ git checkout master
* $ git branch -d feature_x
* $ git push origin <branch name>

<br/>
## Git Merge

* $ git pull (= fetch + merge)
